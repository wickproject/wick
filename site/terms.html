Service Description

Wick is an MCP (Model Context Protocol) server that provides browser-grade web access for AI coding agents. It runs locally on your machine and returns clean markdown content from web pages.

Wick (Free) is open-source software distributed under the MIT License. It includes Chrome-grade TLS fingerprinting, web search, and content extraction.
